# project name: keep-in-touch 

A modern friendship tracking web application that helps users stay connected with friends by managing calls, texts, and video interactions through a clean and responsive interface.

---

## 🚀 Technologies Used

- Next.js
- React.js
- Tailwind CSS
- Context API
- JavaScript

---

## ✨ Key Features

### 1. Timeline Activity Filter
Users can filter timeline activities by:
- 📞 Call
- 💬 Text
- 🎥 Video

---

### 2. Dynamic Status System
Friend status automatically changes with dynamic colors:
- 🔴 Overdue
- 🟡 Almost Due
- 🟢 On-Track

---

### 3. Global State Management
Used React Context API to manage shared data across:
- Navbar
- Timeline
- Friend Details Page

---

## 📱 Responsive Design

Fully responsive across:
- Mobile
- Tablet
- Desktop
